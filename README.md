# SpaceX Launch Cost Prediction Project

## Overview
This project was developed as part of the IBM Data Science program on Coursera. The objective is to analyze and model the cost structure of rocket launches, with a focus on predicting whether a launch can be performed at reduced cost through booster reusability.

## Business Context
The project considers a hypothetical aerospace company aiming to compete with SpaceX. SpaceX is widely regarded as an industry leader due to its ability to recover and reuse the first stage of its rockets, significantly lowering launch costs.

In particular, the Falcon 9 launch system is notable for its comparatively low cost (approximately €62 million per launch), whereas competing systems can cost up to €165 million. This cost advantage is largely driven by the successful reuse of rocket boosters.

## Objective
The primary goal of this project is to build a predictive model that determines whether a rocket booster will be successfully reused. This prediction serves as a proxy for estimating launch costs, enabling strategic decision-making for a competing company.

## Methodology
- Data collection from publicly available SpaceX launch records  
- Data cleaning and preprocessing  
- Exploratory Data Analysis (EDA)  
- Feature engineering
- Interactive Dashboards
- Supervised machine learning models for classification  

## Results
The model provides predictions on booster landing success, which directly impacts cost efficiency. Accurate predictions can support investment decisions and operational planning in the aerospace sector.

## Technologies Used
- Python  
- Jupyter Notebooks  
- Pandas, NumPy  
- Scikit-learn  
- Matplotlib, Seaborn  

## Conclusion
This project demonstrates how data-driven approaches can be used to analyze and predict key factors influencing launch costs. The ability to anticipate booster reuse offers a significant competitive advantage in the commercial space industry.

## Future Work
- Incorporation of additional launch parameters  
- Use of advanced models (e.g., ensemble methods, deep learning)  
- Real-time prediction systems for operational use
